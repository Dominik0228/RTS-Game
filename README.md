\# RTS-Game



Unity version: Unity 6000.0.62f1



\## Controls

\- Flagship Menu - WASD + Shift to run

\- Left click select

\- Right click move

\- Game - Mouse and Arrow keys for camera movement



\## How to Run

1\. Open the project in Unity Hub

2\. Use Unity version 6000.0.62f1

3\. Open the scene: `MainMenu`

4\. Press Play
